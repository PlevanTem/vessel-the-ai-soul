# Long-Term Enoughness

1. **Define enough** — monthly/annual spend that covers values + margin (ranges OK).
2. **Lifestyle creep** — what changed in last 3 years? Was it worth it?
3. **Saving as optionality** — months of runway / dry powder for surprises.
4. **Time horizon** — match assets and career bets to *behavioral* horizon.
5. **One constraint** — rule that protects compounding (automation, cap on spending category).
