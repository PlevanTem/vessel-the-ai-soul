# Independence Geometry

1. **Define enough** — monthly needs + margin (numbers or ranges).
2. **Hours owned** — calendar truth vs. aspiration.
3. **Growth question** — what specifically would 2× scale improve? What would it cost?
4. **Hidden status motive** — would you still want this if invisible?
5. **One change** — smallest move toward the geometry you claim to want.
