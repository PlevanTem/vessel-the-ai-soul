# Hell Yes Filter

1. Name the commitment (time horizon, what you'd actually do day to day).
2. **Body check** — energy at the thought of *doing the work*, not the idea.
3. If the answer is "pretty good / maybe / could be interesting" → treat as **no** for big bets.
4. If **yes**, define the **smallest reversible step** before full commitment.
5. If **no**, practice the **polite decline** — brevity is kindness.
