# Cargo Cult Science Detector — Extended Checklist

*Reference material for `../SKILL.md`*
*Operationalizing Feynman's 1974 Caltech commencement: "Cargo Cult Science"*

## The Checklist (Extended Version)

### 1. Disconfirmation commitment

| Item | Pass | Fail | Notes |
|------|------|------|-------|
| Investigator stated in advance what result would falsify the hypothesis | ✓ | ✗ | Pre-registration is gold standard |
| Alternative hypotheses were explicitly tested (not just mentioned) | ✓ | ✗ | "We considered X but..." is not testing X |
| Negative or null results were sought and reported | ✓ | ✗ | File-drawer rates are high in most fields |
| Investigator would describe what it would take to change their mind | ✓ | ✗ | Ask them; the answer is revealing |

### 2. Completeness of reporting

| Item | Pass | Fail | Notes |
|------|------|------|-------|
| All pre-specified outcomes reported | ✓ | ✗ | Outcome switching is common |
| Non-significant results included | ✓ | ✗ | Significance filter is cargo cult behavior |
| Effect sizes reported with confidence intervals | ✓ | ✗ | p-value alone is not enough |
| Sample excluded reported with reasons | ✓ | ✗ | Exclusion criteria can select for results |
| Protocol available (if pre-registered) | ✓ | ✗ | Check deviation from protocol |

### 3. Controls

| Item | Pass | Fail | Notes |
|------|------|------|-------|
| Control group present and well-matched | ✓ | ✗ | Active controls preferred over placebo-only |
| Double-blind where possible | ✓ | ✗ | At minimum, outcome assessors should be blinded |
| Randomization procedure adequate | ✓ | ✗ | Check for allocation bias |
| N adequate for the claimed effect size | ✓ | ✗ | Use power analysis to check |
| Time horizon appropriate for the outcome | ✓ | ✗ | Surrogate endpoints ≠ real outcomes |

### 4. Replication

| Item | Pass | Fail | Notes |
|------|------|------|-------|
| Independent replication attempted | ✓ | ✗ | Same lab doesn't count as independent |
| Independent replication succeeded with similar effect | ✓ | ✗ | Partial replication is still informative |
| Data and materials shared for replication | ✓ | ✗ | Unavailability is a flag |
| Non-replication explained with testable account | ✓ | ✗ | "They did it wrong" is not testable |

### 5. Conflicts of interest

| Item | Pass | Fail | Notes |
|------|------|------|-------|
| Financial interests disclosed | ✓ | ✗ | Undisclosed is worse than disclosed |
| Effect size consistent with non-industry funded studies | ✓ | ✗ | Industry effect sizes systematically larger |
| Investigator has published contradictory prior findings | ✓ | ✗ | Track record matters |
| Criticism has been responded to with evidence | ✓ | ✗ | Defensiveness is a signal |

## How to Read a Press Release About Science

1. Find the actual paper (not just the press release)
2. Identify: is this an observational study, RCT, meta-analysis, or pre-clinical result?
3. Find the effect size with confidence intervals
4. Check: is the outcome a surrogate marker or a real-world outcome?
5. Check: who funded it?
6. Search for independent replication

**Translation guide:**

| Press release language | What it usually means |
|------------------------|----------------------|
| "Scientists discover X causes Y" | Observational correlation found |
| "New study shows promising results" | Early-stage finding, often pre-clinical |
| "Experts say X" | Not a study; solicited opinion |
| "Breakthrough" | PR term; apply extra skepticism |
| "Statistically significant" | p < 0.05; tells you little about effect size |
| "Could lead to new treatments" | Mechanism hypothesis; many years from application |

## Feynman's Standard (Verbatim)

From "Cargo Cult Science," Caltech, 1974:

"For example, I was a little surprised when I was talking to a friend who was going to tell
a young man the principles of science. He said, 'I am going to give him the essence of
science. When you have a hypothesis and you test it, and it is wrong, you update your model.'
I said, 'Isn't that just the beginning?' Science requires not just that kind of honesty
but a kind of utter honesty — a kind of leaning over backwards. For example, if you're
doing an experiment, you should report everything that you think might make it invalid — not
only what you think is right about it: other causes that could possibly explain your results;
and things you thought of that you've eliminated by some other experiment, and how they worked
— to make sure the other fellow can tell they have been eliminated."

"Details that could throw doubt on your interpretation must be given, if you know them."

"The idea is to try to give all of the information to help others to judge the value of
your contribution; not just the information that leads to judgment in one particular direction."
